operate(a, b, operation) → {string}

  • operate is called in the following situations
  • when = is clicked
  • in this case we also need to check if an operator has been selected and if next value has been populated
  • current total is passed in as first argument
  • next value is passed as second argument
  • in this case both arguments should be non-null
